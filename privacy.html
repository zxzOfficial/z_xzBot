Politique de Confidentialité

Dernière mise à jour : juin 2025

01

Introduction

Cette Politique de Confidentialité décrit comment z_xzbot collecte, utilise et protège les données lors de son utilisation sur Discord. Nous nous engageons à ne collecter que les données strictement nécessaires au fonctionnement du bot.

02

Données collectées

Voici un récapitulatif précis des données que z_xzbot stocke :

Donnée Pourquoi Stockée
ID du serveur (guild) Configuration par serveur ✓ Oui
ID des salons & rôles Logs, tickets, mute, autorole ✓ Oui
ID de l'utilisateur Warns, historique modération, tickets ✓ Oui
Nom d'utilisateur (texte) Affichage dans les logs et embeds ✓ Oui
Contenu des messages Non stocké (traitement en temps réel uniquement) ✗ Non
Mot de passe / e-mail Jamais demandé ✗ Non
Données de paiement Aucune transaction effectuée ✗ Non
03

Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Appliquer la configuration du serveur (rôles, salons, paramètres)
  • Gérer les avertissements et l'historique de modération
  • Faire fonctionner le système de tickets
  • Tracer les actions de modération dans les logs
  • Gérer les giveaways (nom organisateur, salon)

Aucune donnée n'est vendue, partagée ou transmise à des tiers.

04

Stockage et sécurité

Les données sont stockées localement sur le serveur hébergeant le bot, dans des fichiers JSON. L'accès est limité à l'hébergeur du bot. Aucune base de données en ligne ou service cloud tiers n'est utilisé pour stocker vos données.

05

Conservation des données

Les données d'un serveur sont conservées tant que le bot y est présent. Lorsque le bot quitte un serveur, ses données peuvent être supprimées sur demande. Les données d'un utilisateur (warns, notes, historique) peuvent être effacées via les commandes de modération ou sur demande directe au développeur.

06

Vos droits

Conformément aux réglementations applicables, vous disposez des droits suivants :

  • Droit d'accès à vos données stockées par le bot
  • Droit de rectification de données inexactes
  • Droit à la suppression de vos données
  • Droit d'opposition au traitement de vos données

Pour exercer ces droits, contactez le développeur du bot via Discord.

07

Modifications

Cette politique peut être mise à jour à tout moment. La date de révision en haut de cette page sera modifiée en conséquence. L'utilisation continue du bot après modification vaut acceptation de la nouvelle politique.
